KEY FEATURES
================================================================================

1. DATA FETCHING
   - Yahoo Finance integration
   - Real-time price updates
   - Historical data (1 year default)
   - Fundamental company data
   - Multi-stock simultaneous fetching
   - Data caching for efficiency

2. TECHNICAL ANALYSIS
   - Simple Moving Average (SMA): 20, 50, 200 periods
   - Exponential Moving Average (EMA)
   - Relative Strength Index (RSI): 14-period
   - MACD: Trend following indicator
   - Bollinger Bands: Volatility measurement
   - Average True Range (ATR): Volatility measure
   - Stochastic Oscillator: Momentum indicator

3. FUNDAMENTAL ANALYSIS
   - Valuation metrics: P/E, Forward P/E, PEG
   - Financial health: Debt/Equity, Current Ratio
   - Profitability: Profit Margin, ROE, ROA
   - Growth metrics: Revenue Growth
   - Dividend Analysis: Dividend Yield

4. SIGNAL GENERATION
   - Golden Cross detection (SMA20 > SMA50 > SMA200)
   - Death Cross detection (SMA20 < SMA50 < SMA200)
   - RSI Oversold (<30) and Overbought (>70) signals
   - MACD crossovers
   - Price level confirmations
   - Confidence scoring (0-100%)

5. AI/ML CAPABILITIES
   - Random Forest signal classification
   - Gradient Boosting for predictions
   - Feature scaling with StandardScaler
   - Price prediction models
   - Anomaly detection
   - Stock correlation analysis

6. NOTIFICATIONS
   - Email alerts (HTML formatted)
   - Telegram messages
   - Webhook integration
   - Custom notification reasons
   - Timestamp logging

7. USER INTERFACES

8. DOCUMENTATION

USAGE EXAMPLES
================================================================================

1. ANALYZE SINGLE STOCK
   $ python cli.py analyze AAPL

2. ANALYZE MULTIPLE STOCKS
   $ python cli.py analyze AAPL MSFT GOOGL TSLA -p 1y

3. FIND BUY OPPORTUNITIES
   $ python cli.py buy AAPL MSFT GOOGL -c 0.7

4. FIND SELL OPPORTUNITIES
   $ python cli.py sell AAPL MSFT GOOGL

5. FIND HOT STOCKS
   $ python cli.py hot AAPL MSFT GOOGL TSLA AMZN

6. WEB DASHBOARD
   $ streamlit run dashboard.py

   app = StockAnalyzerApp()
   result = app.analyze_single_stock('AAPL')
   buy_opps = app.find_buy_opportunities(['AAPL', 'MSFT'])

INSTALLATION & SETUP
================================================================================

1. Clone/Download project
2. Create virtual environment:
   python -m venv venv

3. Activate environment:
   source venv/bin/activate  # Linux/Mac
   venv\Scripts\activate      # Windows

4. Install dependencies:
   pip install -r requirements.txt

5. Configure settings:
   cp .env.example .env
   # Edit .env with your settings

6. Run application:
   python main.py
   python cli.py analyze AAPL
   streamlit run dashboard.py
